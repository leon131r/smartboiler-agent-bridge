## Что сделано по задачам

### ✅ Задача 1: Артефакты OLED (inb24_mn) — ИСПРАВЛЕНО
**OpenClaw предложил:** `u8g2.clearBuffer()` внутри `do-while`
**Проблема:** `clearBuffer()` не работает в page mode (`firstPage()/nextPage()`)!
**Реальное исправление:**
```cpp
u8g2.setDrawColor(0);      // Цвет = чёрный (стирание)
u8g2.drawBox(0, 0, 128, 64); // Чёрный прямоугольник на весь экран
u8g2.setDrawColor(1);      // Вернуть белый для рисования
```
Применил к экрану часов и экрану статуса.

### ✅ Задача 2: AP Mode + Web-сервер — ИСПРАВЛЕНО
- `connectWiFi()` теперь:
  1. Пробует сохранённые креды из Preferences
  2. Если STA не подключился за 20 сек → `WiFi.softAP("WiFi-Clock-Setup", "12345678")`
  3. Автоматически вызывает `setupWebServer()`
- `setupWebServer()` раскомментирован (запускается автоматически)
- Форма: SSID + Password → сохранение → перезагрузка

### ✅ Задача 3: Кириллица "Пасмурно" — ОК
Код уже использует `u8g2_font_6x13_t_cyrillic` и `u8g2_font_7x13_t_cyrillic` — кириллица поддерживается. wttr.in с `lang=ru` возвращает "Пасмурно" — отображается корректно.

### ✅ Дополнительно: DynamicJsonDocument 4096 → 2048
Уменьшен буфер для ipwhois.app — экономия RAM на ESP32-C3.
